# Shopping-list-manager
A simple command line based shopping list tool. 
The program provides a text-based menu to add, remove, check off, and view items.

## Features
- Add items to your shopping list
- Remove items from the list
- Mark items as completed
- View the full list at any time
- Simple text-based interface

## Getting Started

### Prerequisites
- Java 17+ (or any recent Java version)
- A terminal/command prompt

### Running the Program
1. Clone this repository:
   ```bash
   git clone https://github.com/Canadian599xx/Shopping-list-manager.git
   cd Shopping-list-manager

2. Compile the code:
    ```bash
    javac src/*.java

3. Run the program:
    ```bash
    java -cp src Main
